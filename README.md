# 🍲 Malatang Prediction (麻辣湯 具材提案ツール)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
![Python Version](https://img.shields.io/badge/python-3.8%2B-blue)

「今日の麻辣湯、予算内でどの具材を選べばいいんだろう？」  
そんな悩みを解決する、予算に合わせた最適な具材の組み合わせを提案するツールです。

## 🌟 概要

麻辣湯（マーラータン）は具材の選び方で無限のバリエーションがありますが、ついつい取りすぎて予算オーバーになりがちです。  
このツールは、ユーザーが指定した予算（Budget）に合わせて、おすすめの具材セットを自動で計算・提案します。

## ✨ 主な機能

- **予算別提案**: 500円〜2000円など、設定した予算に収まる最適な組み合わせを提示。
- **バランス調整**: タンパク質多め、ヘルシーなどのバランスを考慮した提案。
- **簡単操作**: 予算を入力するだけで、瞬時に具材リストを表示。
